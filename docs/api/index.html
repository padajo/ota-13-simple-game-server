API

Game List

GET

/game/api/<client_pk>/games/

Get the list of all games for the given client pk - player list is essentially the same as the player list below

Sample Response

[
   {
      "pk":1,
      "client_pk":1,
      "name":"Game 1",
      "players": []
   },
   {
      "pk":2,
      "client_pk":1,
      "name":"Game 2",
      "players": []
   },
   {
      "pk":3,
      "client_pk":1,
      "name":"Game 3",
      "players": []
   }
]

Player Info

GET

/game/api/<client_pk>/games/<game_pk>/players/<player_pk>/

Get the details for the given user

Sample Response

{
   "pk":2,
   "game_pk":1,
   "items":[
      {
         "player_pk":2,
         "pk":1,
         "amount":1234,
         "name":"1234"
      },
      {
         "player_pk":2,
         "pk":2,
         "amount":213,
         "name":"gwrg"
      }
   ],
   "name":"a",
   "device_id":"12343"
}

Player Join

POST

/game/api/<client_pk>/games/<game_pk>/players/<player_pk>/add/

Variables: name - name of the user and device_id the unique device id

Add a user to the given game

Sample Response

{
   "pk":6,
   "game_pk":1,
   "items":[

   ],
   "name":"a3",
   "device_id":"1234521"
}

Player Remove

POST

/game/api/<client_pk>/games/<game_pk>/players/<player_pk>/delete/

Variables: device_id the unique device id

Remove a user to the given game

Sample Response

{"success": true}

Player Item Add

POST

/game/api/<client_pk>/games/<game_pk>/players/<player_pk>/items/add

Variables: name of the item - amount amount of the item

Add an item to the player

Sample Response

This is the item information

{"player_pk": 2, "pk": 4, "amount": "1", "name": "abc3"}

Player Item Update

POST

/game/api/<client_pk>/games/<game_pk>/players/<player_pk>/items/<item_name>/update

Variables: amount amount of the item

Update the specified item for the player

Sample Response

This is the item information

{"success": true}
